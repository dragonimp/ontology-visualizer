## 4. 3D 渲染升级详情

### 4.1 布局算法升级

| 对比项 | 旧版（2D 平铺） | 新版（3D 球形） |
|--------|----------------|----------------|
| 排列方式 | 根类排成一行，子类 Y+15 递增 | 根类在球心，各层按深度均匀分布在球面上 |
| 算法 | 简单 BFS 队列 | Fibonacci Sphere 均匀分布算法 |
| 半径计算 | 固定 20 像素间距 | `深度 × 65` 球壳半径 |
| 孤立类 | 网格排列 | 外围球面上均匀分布 |

### 4.2 光照系统

- **环境光**（0x303050, 强度 0.8）：基础照明
- **主方向光**（0xffffff, 强度 1.5）：模拟顶光，带阴影
- **蓝色补光**（0x4FC3F7）：左侧冷光，增强科技感
- **紫色补光**（0x9C27B0）：右侧暖光，增加层次
- **底部反光**（0x1a237e）：底部冷色反光

### 4.3 节点材质增强

- **SphereGeometry 精度**：16→24 细分，更圆润
- **TorusGeometry**：8×16→16×32 细分，更光滑
- **高光材质**：shininess=80, specular=0x444444
- **发光增强**：emissiveIntensity 0.2→0.3
- **光晕效果**：半透明 BackSide 球面光晕（0.08 不透明度）

### 4.4 背景与氛围

- **深空背景**：0x0a0a1a
- **粒子星空**：1500 个粒子，蓝白到淡紫渐变，分布在 800-2000 半径球面上
- **雾效**：FogExp2 浓度 0.0008，远处渐变消失
- **自动旋转**：scene.rotation.y += 0.0003，营造沉浸感
- **移除网格**：替换为星空粒子

### 4.5 边连接优化

| 对比项 | 旧版 | 新版 |
|--------|------|------|
| 连接方式 | 直线 | CubicBezierCurve3 贝塞尔曲线 |
| 弧线高度 | 无 | `max(距离 × 0.15, 5)` |
| 类关系边 | 灰色 0.8 不透明度 | 蓝色 0x4FC3F7, 0.4 不透明度 |
| 属性边 | 灰色 0.8 不透明度 | 紫色 0x666688, 0.25 不透明度 |
| 曲线点数 | 2 点 | 20 个采样点 |
