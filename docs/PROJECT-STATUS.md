## 3D 渲染效果总结

### 布局算法
- **Fibonacci Sphere 均匀分布**：同类节点按深度均匀分布在球面上
- **根类在球心**，子类按深度逐层向外扩散
- **孤立类**放在外围球面上

### 视觉效果
- **5 光源系统**：环境光 + 方向光 + 蓝/紫/底部补光
- **高光 Phong 材质**：shininess=80，节点有明显高光反射
- **光晕效果**：类节点有半透明光晕
- **1500 粒子星空**：蓝白到淡紫渐变
- **贝塞尔弧线连接**：替代直线，更有层次感

### 技术参数
| 参数 | 值 |
|------|-----|
| 节点精度 | Sphere 24×24, Torus 16×32 |
| 星空粒子 | 1500 个 |
| 雾效浓度 | 0.0008 |
| 自动旋转 | 0.0003 rad/frame |
| 弧线采样点 | 20 点 |
